# dotfiles

Arch Linux + Hyprland setup

## Stack
- WM: Hyprland
- Bar: Waybar
- Terminal: Kitty
- Shell: Zsh + Powerlevel10k
- Launcher: Rofi
- Notifications: Dunst
- Wallpaper: Waypaper + Hyprpaper
- Lock: Hyprlock
- Power: Wlogout

## Install
git clone https://github.com/yourusername/dotfiles
cd dotfiles
./install.sh
